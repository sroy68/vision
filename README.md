# Vision Security AI

Vision is an analysis-only security platform for bug bounty,
secure coding, and legal CTF practice.

## What Vision Does
- Scans public websites (GET only)
- Analyzes source code statically
- Identifies security risks (XSS, SQLi, headers, secrets)
- Provides fix recommendations

## What Vision DOES NOT Do
- No exploitation
- No payload execution
- No hacking or bypassing
- No data modification

## Modes
### Public PWA
- Free
- Safe analysis-only
- No exact exploit details

### Enterprise
- Private org access
- Detailed findings
- Compliance-ready reports
- Audit logs
